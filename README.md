# Task Manager

Простое приложение для управления задачами с drag-and-drop.

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/murashko12/task-manager.git
```

2. Перейдите в папку проекта:
```bash
cd task-manager
```

3. Установите зависимости:
```bash
npm install
```

## Запуск

Для разработки:
```bash
npm run dev
```

Для сборки:
```bash
npm run build
```
